# Intelligent Spy Car for Voice Surveillance

> **Abstract** : This project is about building an obstacle-avoiding robotic car with a Mini A8 voice recorder for live audio monitoring.
The car moves on its own using ultrasonic sensors to detect obstacles and avoid them automatically. A microcontroller reads the sensor data and controls the motors to help the car move safely without human control.
A Mini A8 voice recorder is attached to the car to record and monitor sounds around it. It supports a SIM card, so you can call the device and listen to the live audio remotely. It also has a sound-detection feature that reacts when noise is detected nearby.
This project combines robotics and audio surveillance, making it useful for security monitoring, indoor patrolling, restricted area surveillance, and learning purposes. It is a simple and low-cost autonomous surveillance system.
